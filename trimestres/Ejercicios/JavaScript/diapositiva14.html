Declaración de variables

JavaScript permite declarar variables usando var, let y const. Aunque var es la forma más antigua, actualmente se recomienda usar let para variables que cambian de valor, y const para constantes que no deben modificarse.

Ejemplos

let edad = 25; const nombre = "Laura"; // let permite reasignar valores let contador = 1; contador = 2; // válido // const no permite reasignación const PI = 3.14; PI = 3.1416; // Error
